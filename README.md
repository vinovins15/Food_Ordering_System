# Food Ordering System

## 📌 Project Description
A full-stack web application for online food ordering. Users can browse menu items, add items to cart, and place orders.

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ✨ Features
- Menu display
- Add to cart functionality
- Order placement
- Database integration
